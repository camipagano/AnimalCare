# AnimalCare
Progetto TSW: AnimalCare si propone di essere un sito di e-commerce focalizzato sulle necessità per gli  appassionati di animali. Ha l’obiettivo di fornire servizi che variano per tutte le specie di  animali domestici con prodotti che possono essere semplice cibo o anche accessori e  strumenti di toelettatura.
